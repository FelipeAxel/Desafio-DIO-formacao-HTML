Tags de titulo

(Voltar)

As tags de h1 ate h6 são tags para criação de títulos em páginas HTML com níveis de importância diferentes para cada h.
Sendo h1 o mais importante e assim vai seguindo até o h6 que não é menos importante que os demais, vale lembrar que tenha apenas 1 h1 por páginas, mesmo que possa mais de 1.
Não é uma boa prática pular a sequência, sempre siga do h1, h2, h3 até h6, e também so vamos usar as tags para título outros sem importância você pode estilizar no css.

Tag de paragrafo

(Voltar)

A tag p serve para separar os parágrafos do seu texto.
Parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.
Eles são elementos block-level, fecharão automaticamente se outro block-level inicie antes da tag de fechamento.

Tag de mark

(Voltar)

Serve para dar destaque em um trecho importante do seu texto ou até mesmo marcar letras de uma pesquisa especifica.
Você pode se perguntar, mas é o mesmo que o strong, mas não, o elemento strong é usado para marcar partes importantes, especiais do texto, o elemento mark é usado para partes de relevância especial.

Tag de small

(Voltar)

A tag small tem como finalidade diminuir a frase ou palavra para o padrão de tamanho small.
Mais usadas em caso de copyright, avisos, regras, restrições legais, direitos de uso.

Tag de i

(Voltar)

A tag i serve para destacar termos técnicos, expressões de outros idiomas ou pensamento de personagens fictícios.
Nas primeiras versões das especificações HTML, a tag i era usada apenas para apresentar textos em itálico, assim como a tag b tag era usada para mostrar textos em negrito.
Isso não é mais verdade, já que, atualmente, essas tags definem a semântica, e não mais aparência tipográfica.
A tag i pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo itálico.
Isso significa que os navegadores continuarão a apresentar o conteúdo em itálico, mas, conforme o definido, isso não é mais requerido.

Tags de lista

(Voltar)

tag ol

A tag ol, significa lista ordenada ou order list representa uma lista de itens ordenados, que pode ser de qualquer tipo, como numerais, letras, algarismos, romanos, ou simples símbolos.

tag ul

A tag ul representa uma lista não ordenada ou order list representa uma lista de itens sem uma ordem rígida.
Mais usada em uma lista que não teria importância em qual sequencia ela seja seguida, como uma lista do supermercado, agora, por exemplo, para uma receita de bolo precisamos seguir corretamente os passos em uma lista ordenada.

tag li

No html o li é usado para representar um item de uma lista do html, mais usadas entro dos elementos pai, ol ou ul.

Tag de a

(Voltar)

A tag a serve como uma ancora que segue como atributo o href e cria-se uma hiperligação nas páginas web, endereço de e-mails. A URL para a qual o hiperlink aponta. Links não se restringem a URLs baseadas no protocolo HTTP — eles podem utilizar qualquer tipo de URL suportado pelos browsers.

Tag de hr

(Voltar)

A tag hr representa uma quebra de linha mais bonita entre os elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).

Tag de sub

(Voltar)

A tag sup serve para definir um texto subscrito, como, por exemplo, a descrição de um elemento químico H2O.

Tag de sup

(Voltar)

A tag sup serve para definir um texto sobrescrito serve para marcar os lugares de um pódio 1°2°3°.

Tag de blockquote

(Voltar)

A tag blockquote, serve para elementos de citação longa como artigos longos de blog, uma crítica de algum restaurante ou filme.
